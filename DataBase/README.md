# Database-Management-System
Database Management System Created using JAVA. To run:
<ol>
  <li>"cd src"</li>
  <li>"javac Database.java"</li>
  <li>"java Database"</li>
  <br>
  <li>Enter command:</li><ul>
    <li>"create table " tablename (value)</li>
   <li>"select * from " tablename </li>
  </ul>
</ol>
